Mode d'emploi : Après une partie, le casino révèle le server seed. Collez-le dans la première section avec le hash affiché avant votre mise pour vérifier que le casino n'a pas changé de seed. Utilisez la deuxième section pour recalculer le résultat HMAC et vérifier qu'il correspond au résultat de votre partie.
